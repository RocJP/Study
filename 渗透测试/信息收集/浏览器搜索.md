# 1、双引号
    双引号里的搜索内容是完全匹配搜索，适用于百度和谷歌。
# 2、减号
    减号代表搜索不包含减号后面的词的页面。适用于百度和谷歌。
# 3、星号
      星号*是常用的通配符，也可以用在搜索中。适用于谷歌。
